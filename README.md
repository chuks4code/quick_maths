# quick_maths

Quick Maths is a Flutter mobile app that challenges users to solve a set of arithmetic questions and provides instant feedback on performance.  
The app showcases clean UI, state management, reusable widgets, and basic evaluation logic — all built using Flutter and Dart.

---

##  Features

-  Generates 4 random math questions per round  
-  Allows users to submit answers and receive feedback  
-  “Play Again” flow with state toggling  
-  Friendly UI with visual button feedback  
-  Clean architecture with reusable components  

---

##  Key Technical Highlights

### Flutter Best Practices
- Reusable `GameButton` with ripple & disabled states  
- Separation of logic and UI  
- State handling for button enabling/disabling  
- Simple but extendable scoring and logic structure

### UI / UX Improvements
- Button effects to improve mobile usability  
- Background branding with subtle logo watermark  
- Responsive input validation before submission
